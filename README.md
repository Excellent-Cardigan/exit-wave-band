## The Concept

The website is being designed as an immersive experience inspired by early 2000s Flash sites — TOOL, Radiohead W.A.S.T.E., 2Advanced. Visitors arrive at a signal, not a landing page.

The centerpiece is an **interactive stem player**: each track's stems are floating nodes in a physics-driven graph. Users connect and disconnect stems, adjust volume per node, and attach beat-reactive visual/animation nodes to the mix. Not just a way to hear the music — a way to interact with it.
